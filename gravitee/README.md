# Gravitee Helm Chart

**Chart supported versions: 1.19.3 and higher**

## Chart Details

This chart will deploy the following:

- Gravitee UI
- Gravitee API
- Gravitee Gateway
- MongoDB replicaset (optionally)
- Elasticsearch cluster (optionally)

## Create a chart archive

To package this chart directory into a chart archive, run:

```
$ helm package .
```

## Installing the Chart

To install the chart using the chart archive, run:

```
$ helm install gravitee-1.23.2.tgz
```

To install the chart with the release name `my-release`:

```bash
$ helm install --name my-release gravitee
```
